$('.head').headroom({'hiddenTop':400,autoHide:false,'scrollOffset':60});

调用方法：

$(selector).headroom(options);

options参数

参数 默认值 说明
fixedTop 0 浮动距离浏览器视窗顶部的位置
hiddenTop [自动计算] 滚动到哪个位置开始隐藏菜单
duration 500 隐藏展示动画的时间间隔
autoHide true 是否自动隐藏
scrollOffset 0 描点滚动偏移顶部距离
activeCls 'active' 菜单选中时的class
