系统默认颜色选择扩展

悬浮时实时改变

$('input[type="color"]').colorpicker({hoverChange:true});

文本输入框颜色选择扩展

修改完成后触发事件

$('input[type="text"]').colorpicker({
	'onSelect':function(color){
		$('#test').text('颜色修改为：'+color);
	}
});

调用方法：

$(selector).colorpicker(options);

options参数

参数 默认值 说明
id 'colorpicker' 颜色选择器的ID
size 'default' 选择器大小；default默认大小，small小号
hoverChange false 悬浮于调色板时是否立即触发修改表单值
onChange [空] 悬浮于调色板时触发
onSelect [空] 选中调色板时触发
